# 🎵 Music Data Analysis (Spotify)

> Descubrir qué hace popular a una canción y agrupar temas por "vibes" similares.

**Estado:** 🚧 En construcción — ver [`ROADMAP.md`](ROADMAP.md) para el plan completo por fases.

## 🎯 Objetivo
EDA, análisis por género y **clustering (K-Means + PCA)** sobre el dataset de Spotify Tracks
para encontrar patrones de audio asociados a la popularidad y agrupar canciones sin usar el
género como etiqueta.

## 🛠️ Stack previsto
Python · pandas · scikit-learn · matplotlib / seaborn · plotly

## 📄 Dataset
[Spotify Tracks Dataset — Kaggle](https://www.kaggle.com/datasets/maharshipandya/-spotify-tracks-dataset)
(~114,000 canciones; no incluido en el repo — ver instrucciones en el ROADMAP).
